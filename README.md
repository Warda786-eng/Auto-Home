# AutoHome — Smart Home Dashboard

A fully interactive Smart Home Dashboard built with HTML, CSS and JavaScript. 
Control all your home devices from one place, set automation rules and monitor 
your energy usage in real time.

## Live Demo
[Click here to view AutoHome](https://yourusername.github.io/autohome)

## Features

- Login system with secure access
- 5 pages — Dashboard, Rooms, Automations, Energy, Security
- 12 smart devices across 6 rooms
- Device toggles — turn lights, fans, AC and more on or off
- Sliders for brightness, AC temperature, fan speed and volume
- 6 automation rules including Night Mode and Climate Control
- Night Mode automatically turns off all lights at 10PM
- Weekly energy chart showing solar vs consumed power
- Security page with camera feeds and door lock controls
- Live clock and weather widget
- Fully mobile responsive with bottom navigation

## How to Use

1. Open the dashboard in any browser
2. Login with username admin and password 1234
3. Control your devices from the Dashboard or Rooms page
4. Set automation rules on the Automations page
5. Monitor power usage on the Energy page
6. Check cameras and locks on the Security page
